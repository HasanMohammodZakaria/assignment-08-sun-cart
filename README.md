# 🌞 SunCart – Summer Essentials Store

## 📌 Project Purpose
SunCart is a modern summer-themed eCommerce web application where users can explore and purchase seasonal products like sunglasses, outfits, skincare, and beach accessories.  
The goal of this project is to build a responsive and user-friendly online store with authentication and protected routes using Next.js.

---

## 🚀 Live URL
👉https://sun-cart-peach.vercel.app

---

## ✨ Key Features
- 🔐 User Authentication (Login & Register with Email/Password)
- 🔑 Google Social Login
- 🔒 Protected Product Details and My Profile Page
- 🛍️ Browse Summer Products
- 📄 View Detailed Product Information
- 👤 User Profile Page
- ✏️ Update User Profile (Name & Image)
- 📱 Fully Responsive Design (Mobile, Tablet, Desktop)
- 🎨 Modern Summer UI Design
- 🔔 Toast Notifications for Success/Error
- ⏳ Loading Spinner for Better UX
